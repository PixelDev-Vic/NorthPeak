# NorthPeak ERP — Homepage Prototype

Static homepage for the NorthPeak Enterprise Resource Planning system.
Built with semantic HTML5 and Bootstrap 5 (CDN) for responsive layout — no build step, no framework.

**To view:** open `index.html` in any modern browser. No install or server required.

**Design notes:** The typeface is *Space Grotesk* (Google Fonts) — a geometric sans-serif with a technical, modern feel suited to enterprise software. The color palette avoids Bootstrap defaults entirely, using a five-tone blue scale ranging from near-black navy (`#0f1e2e`) through a barely-blue frost (`#f0f4f8`) to white, creating depth and hierarchy without relying on saturated accent colors.
